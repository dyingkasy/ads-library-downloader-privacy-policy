Política de Privacidade

Ads Library Downloader

Última atualização: 27 de abril de 2026

Resumo

O Ads Library Downloader não coleta, vende, compartilha ou transmite dados pessoais do usuário para servidores externos.

Como a extensão funciona

A extensão funciona localmente no navegador. Quando o usuário clica para detectar criativos, a extensão lê apenas o conteúdo visível da aba ativa, como imagens, vídeos, links e textos disponíveis na página aberta.

Quando o usuário clica para baixar, os arquivos detectados são salvos localmente na pasta de downloads do navegador. A extensão também pode gerar um arquivo metadata.json contendo URL de origem, título da página, links detectados, textos visíveis dos anúncios e eventuais erros de download.

Dados processados localmente

  • URLs de imagens, vídeos e links visíveis na página aberta.
  • Textos visíveis relacionados aos blocos de anúncio detectados.
  • Título e URL da página onde o usuário executou a extensão.
  • Erros de download registrados no arquivo metadata.json.
  • Preferências locais de organização, como pasta base, prefixo e opções de agrupamento escolhidas pelo usuário.

Compartilhamento e transmissão

Essas informações não são enviadas para servidores externos pela extensão. O processamento ocorre localmente no navegador do usuário e os arquivos são salvos apenas quando o usuário solicita o download.

As preferências de organização são salvas localmente no navegador do usuário por meio do armazenamento da extensão. Essas preferências não são transmitidas para servidores externos.

O que a extensão não faz

  • Não usa cookies próprios.
  • Não rastreia a navegação do usuário.
  • Não coleta credenciais, senhas ou dados de login.
  • Não coleta dados financeiros ou informações de pagamento.
  • Não vende, compartilha ou transfere dados do usuário.
  • Não utiliza código remoto.

Contribuições voluntárias via Pix

A extensão pode exibir uma chave Pix ou QR Code para contribuições voluntárias. A extensão permanece gratuita e nenhuma funcionalidade é bloqueada por pagamento.

A extensão não processa pagamentos, não coleta dados financeiros e não recebe informações sobre quem realizou uma contribuição. Qualquer operação Pix é feita diretamente pelo usuário no aplicativo do seu banco ou instituição de pagamento.
